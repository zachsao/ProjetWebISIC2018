Nom du projet : Carcar Binks

Auteurs et copyright : Zacharia Sao & Pierre Galus

État du projet : Fonctionnel mais imcomplet (manque de temps)

Description : Ce projet est un site web de covoiturage qui a pour vocation d'être utilisé par les élèves et agents de l'école d'ingénieur IMT Lille Douai.

Utilisation :

Vous arrivez par défaut sur la page index.
Vous pouvez naviguer à travers les différentes pages via la barre de naviguation située en haut du site.
Vous pouvez consulter les trajets proposés sans vous connecter. Cependant, une connexion est nécessaire pour rejoindre ou créer un trajet.
Une fois connecté, vous avez accès à votre compte où sont répertoriées toutes vos informations personnelles.
Vous pouvez aussi accéder à la page décrivant qui nous sommes en bas du site.

Si vous êtes un administrateur du site, il vous sera possible de supprimer un trajet, supprimer un utilisateur, promouvoir au rang d'administrateur un utilisateur ou destituer un administrateur au rang d'utilisateur normal.


Astuces : 
Vous pouvez vous connecter en tant qu'utilisateur commun avec les identifiants suivants : 
Login : jarjar
Mot de passe : jarjar 

Vous pouvez vous connecter en tant qu'administrateur avec les identifiants suivants : 
Login : root
Mot de passe : root 

Toutes les requetes SQL sont dans le dossier : sql/requetes.txt
